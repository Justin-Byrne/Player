# .MDJ File Extension

## What is a MDJ file?
- Mdj file extension has StarUML Model file type, developed by MKLab, which is associated with the "data files" category.

	**Extension Details**
    
    | Developer: | MKLab              |
	|------------|--------------------|
	| File Type: | StarUML Model File |
	| Category:  | Data Files         |
	| Format:    | Text               |
